# JTX Video Workflow Checklist

## 1. 素材预处理

- [ ] 原始视频已确认
- [ ] 已生成 30fps 固定关键帧视频
- [ ] 已提取音频
- [ ] 已记录素材路径

## 2. SRT

- [ ] 已转写
- [ ] 已输出 SRT 文件
- [ ] 已给用户审核
- [ ] 用户已确认 SRT

## 3. 内容结构

- [ ] 已拆分文案类型
- [ ] 已保存结构分析文件
- [ ] 已给用户审核
- [ ] 用户已确认结构

## 4. 分镜表

- [ ] 已生成分镜表
- [ ] 每个镜头都有画面策略
- [ ] 已避免单调重复
- [ ] 已提炼真正关键的 5 到 8 个视觉重点
- [ ] 已删除非核心、重复、装饰性的文字展示
- [ ] 关键数字/结论已绑定到对应 SRT 时间点
- [ ] 已给用户审核
- [ ] 用户已确认分镜

## 5. 设计

- [ ] 已从默认模板生成项目 `design.md`
- [ ] 已确认源视频方向，横屏不默认转竖屏
- [ ] 已确认人物亮度规则
- [ ] 已确认字幕安全区
- [ ] 已确认动效禁忌
- [ ] 已确认无素材时用动态重点字，有素材时真实素材优先
- [ ] 已确认重点字为实心、粗体、高对比，不用纹理/折叠/镂空效果
- [ ] 用户已确认设计或沿用默认

## 6. HyperFrames 工程

- [ ] 使用连续主视频
- [ ] 视频与音频分轨
- [ ] 字幕层完整
- [ ] 重点字/素材层完整
- [ ] 非必要卡片已删除
- [ ] 重要素材完整显示，没有裁断、遮挡或重复标签
- [ ] 横屏资料讲解段采用左素材、右人物同步布局
- [ ] 横屏分屏背景是静态虚化帧，不是动态人物背景
- [ ] PIP 人物视频使用独立稳定素材或根级视频层
- [ ] 无花哨禁用动效

## 7. 验证

- [ ] `npx hyperframes validate`
- [ ] `npx hyperframes inspect --samples 12`
- [ ] 预览帧 / 九宫格已生成
- [ ] 对用户反馈点使用 `npx hyperframes snapshot --at ... --describe false` 精确抽帧
- [ ] 已检查重点数字出现前一帧和出现后一帧
- [ ] 已检查分屏段只有人物小窗在动，背景不动
- [ ] 已检查字幕不与素材/人物容器重叠
- [ ] 音频未截断

## 8. 导出

- [ ] 审核版已生成
- [ ] 按源方向生成母版
- [ ] 最终 2K60 已生成
- [ ] `ffprobe` 已确认规格
- [ ] `ffprobe` 已确认视频/音频时长接近
- [ ] 最终 MP4 已抽关键帧复查
- [ ] 已向用户报告最终路径
