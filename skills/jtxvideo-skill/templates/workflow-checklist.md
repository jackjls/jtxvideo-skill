# JTX Video Workflow Checklist

## 1. 素材预处理

- [ ] 原始视频已确认
- [ ] 已生成 30fps 固定关键帧视频
- [ ] 已提取音频
- [ ] 已记录素材路径

## 2. SRT

- [ ] 已转写
- [ ] 已输出 SRT 文件
- [ ] 已给用户审核
- [ ] 用户已确认 SRT

## 3. 内容结构

- [ ] 已拆分文案类型
- [ ] 已保存结构分析文件
- [ ] 已给用户审核
- [ ] 用户已确认结构

## 4. 分镜表

- [ ] 已生成分镜表
- [ ] 每个镜头都有画面策略
- [ ] 已避免单调重复
- [ ] 已给用户审核
- [ ] 用户已确认分镜

## 5. 设计

- [ ] 已从默认模板生成项目 `design.md`
- [ ] 已确认人物亮度规则
- [ ] 已确认字幕安全区
- [ ] 已确认动效禁忌
- [ ] 用户已确认设计或沿用默认

## 6. HyperFrames 工程

- [ ] 使用连续主视频
- [ ] 视频与音频分轨
- [ ] 字幕层完整
- [ ] 卡片层完整
- [ ] 无花哨禁用动效

## 7. 验证

- [ ] `npx hyperframes validate`
- [ ] `npx hyperframes inspect --samples 12`
- [ ] 预览帧 / 九宫格已生成
- [ ] 音频未截断

## 8. 导出

- [ ] 审核版已生成
- [ ] 1080p60 母版已生成
- [ ] 最终 2K60 已生成
- [ ] `ffprobe` 已确认规格
- [ ] 已向用户报告最终路径
